1920 Fluid grid system

Version: 1.0 beta
Released: 14/5/2012
Distributed as part of Mainframe PHP framework

Author: Vangelis Bibakis
Twitter: @bibakis

How to use it

			Div style:

			<div class="row">

				 <div class="grid_4">

					  Some text here

				 </div>

				 <div class="grid_8">

					  And some more

				 </div>

			</div>


			Clear style:

			

				<div class="grid_4">

					 Some text here

				</div>

				<div class="grid_8">

					 And some more

				</div>

			<div class="clear"></div>

			

			.clear {clear:both;}

Also...

Use gridX for simple boxes, and grid_X for boxes with margins

We recommend that you use 1920.gs along with a nice CSS reset. We ship Mainframe PHP with html5 boilerplate's reset file and recommend it.


Known limitations:

Due to the way that browsers calculate percentages you won't be able to achieve pixel perfect layouts across all browsers. This a technical browser limitation and not a 1920 specific issue. I chose to ignore this, since in my opinion, the benefits of a minimal fluid layout codebase far outweight such issues.
